RAM - Random Access Memory

Test

Volatile piece of hardware that stores live memory.

When you cut the power to RAM all data is lost! It is very fast to be accessed when on.

Different RAM speeds are measured in Megahertz (MHz)

The two main types of RAM are Dynamic RAM (DRAM) and Static RAM (SRAM). DRAM is widely used as computer main memory, while SRAM is often used for cache memory. Within DRAM, there are also variations like Synchronous Dynamic RAM (SDRAM) and Double Data Rate Synchronous Dynamic RAM (DDR SDRAM), like DDR2-5.

ROM - Read Only Memory

Non-volatile piece of hardware the stores basic instruction such as BIOS.

It also stores: Initial boot instructions, hardware initialization routines, and self-test procedures

While ROM is very difficult to overwrite it is possible to change it but that is not recommended. You should just get a new chip with what you want in it already.
