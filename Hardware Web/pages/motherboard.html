Motherboards

A motherboard is the main circuit board of a computer. It connects and all the major components like the
CPU, RAM, GPU, storage devices, and power supply. It also allows them to communicate to each other.

It houses the CPU via a socket, holds RAM slots, provides PCIe slots for GPUs and expansion cards, offers SATA / M.2 ports for SSDs and hard drives,
includes USB, audio, and network ports, distributes power to components.

There are three main form factors. They are:
ATX - 12" x 9.6" - Used for full-size builds, and it has most features
Micro-ATX (mATX) - 9.6" x 9.6" - Is in smaller cases, and has fewer slots
Mini-ITX - 6.7" x 6.7" - Used in compact PCs, and has limited expandability.
