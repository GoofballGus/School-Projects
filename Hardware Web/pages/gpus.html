GPU - Graphics Processing Unit

A GPU is a specialized processor designed to render images, videos, and animations. It handles complex visual and mathematical tasks,
offloading them from the CPU increasing computer operating speed. GPUs are used for gaming, video editing, 3D rendering, and AI tasks.

There are two types of GPU. They are:
Integrated GPU (iGPU): They are built into the CPU and shares system memory (RAM). They are used for basic tasks as web browsing, video streaming, light gaming.
Dedicated GPU (Discrete GPU): This is a separate graphics card with its own VRAM (Video Random Access Memory). It delivers high performance for gaming, 3D rendering, and compute workloads.

There are 5 main parts to a GPU:
VRAM: Memory used by the GPU (4GB–24GB+ common)
Core Clock: Speed of the GPU’s cores (MHz/GHz)
CUDA / Stream Cores: Parallel processors (NVIDIA = CUDA, AMD = Stream)
TDP: Power draw (affects PSU & cooling needs)
Bus Interface: Typically PCIe x16 (check for compatibility)
