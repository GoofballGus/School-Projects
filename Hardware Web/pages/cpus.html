CPU - Central Processing Unit

The CPU (Central Processing Unit) is often called the “brain” of the computer. It performs most of the processing tasks, running your operating system,
apps, and handling all general instructions your PC needs to function.
The CPU then uses the Fetch-Decode-Execution which uses the data to perform the instructions for the app.

There are six parts needed for a CPU:
Cores - Independent processing units (more cores = better multitasking)
Threads - Virtual cores (thanks to SMT/Hyper-Threading)
Base/Boost Clock - Speed in GHz; higher = faster, but not always better alone
Cache - Fast memory inside the CPU for quick access
TDP - Power consumption (affects cooling and PSU needs)
Socket Type - Physical connector on motherboard (must match motherboard)
